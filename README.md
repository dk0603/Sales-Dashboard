# Sales Performance Dashboard - Global Superstore

## Project Overview
This Power BI dashboard provides an interactive analysis of the sales performance data from the Global Superstore dataset. It is designed to help businesses understand key sales trends, product performance, and customer behavior.

## Key Features
- **Sales Overview**: Display of total sales, profits, and quantities sold over a selected time range.
- **Category Analysis**: Breakdown of sales and profits by product categories and sub-categories.
- **Customer Insights**: Analysis of customer segments and regions, identifying key markets and customer behavior patterns.
- **Profitability**: Insights into the profitability of different products and regions.

## Visualizations
- KPIs for Total Sales, Profit, and Quantity Sold.
- Line chart for Sales Trends over time.
- Bar charts for top products and regions.
- Map visualization for geographical sales distribution.
- Pie charts for product category breakdown.
- Scatter plot for Sales vs. Profit analysis.

## How to Use
1. Clone this repository to your local machine.
2. Open the Power BI file (`Sales_Performance_Dashboard.pbix`) in Power BI Desktop.
3. Use the interactive slicers (Date Range, Category, Region) to explore different aspects of the dataset.

## Dataset
This project uses the Global Superstore dataset. You can download the dataset [here](https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting).
